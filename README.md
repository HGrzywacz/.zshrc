.zshrc - cloning
======
-------------------------------

1. Install oh-my-zsh:

        sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"


2. Clone and link this config:

        $ git clone https://github.com/HGrzywacz/.zshrc.git ~/.zsh
        $ ln -s ~/.zsh/zshrc ~/.zshrc
